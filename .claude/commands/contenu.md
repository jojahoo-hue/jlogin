# Commande /contenu

> Génération de contenu pour l'univers artistique, spirituel et entrepreneurial de Njaho.
> Style : initiatique, géo-mathématique, tradition Congo, élévation de conscience.

---

## Modes disponibles

```
Création de contenu — quel type ?

1. /contenu nft        → Description et pitch d'une oeuvre NFT
2. /contenu livre      → Rédaction d'un passage / chapitre initiatique
3. /contenu post       → Post réseaux sociaux (Instagram, X, LinkedIn)
4. /contenu site       → Texte pour le site artistique
5. /contenu pitch      → Pitch de présentation de l'univers artistique
```

---

## Style de référence — À appliquer dans tous les modes

L'écriture de Njaho est :
- **Initiatique et littérale** : chaque mot porte un sens profond, rien n'est décoratif
- **Ancrée dans la tradition Congo** : références aux forces cosmiques, aux énergies élémentaires, aux lignées ancestrales
- **Géo-mathématique** : les formes naissent de formules, la géométrie est sacrée
- **Universelle** : accessible à tous mais à plusieurs niveaux de lecture
- **Sobre et dense** : pas de fioriture, chaque phrase est une pierre posée

Ne jamais écrire dans un style poétique creux ou ésotérique vague.
Toujours ancrer dans le concret : une forme, une couleur, une formule, un ancêtre, une énergie.

## Mode 1 : /contenu nft

### Étape 1 — Informations sur l'oeuvre

```
Décris-moi l'oeuvre :

1. Titre (ou titre provisoire)
2. Support et technique (toile, numérique, format)
3. Couleurs dominantes et formes présentes
4. Formule géo-mathématique ou code de fréquence à la base (si applicable)
5. Ce que tu as voulu transmettre en créant cette oeuvre
```

### Étape 2 — Générer les textes NFT

Produire trois éléments :

**Description courte (max 150 caractères)**
Pour l'aperçu sur la marketplace NFT.

**Description longue (300-500 mots)**
Dans le style initiatique de Njaho. Structure :
- L'origine (d'où vient cette oeuvre, quelle intention)
- La forme (ce que la géométrie exprime)
- L'énergie (ce que le regardeur reçoit)
- La lignée (lien avec la tradition Congo, les civilisations africaines)

**Attributs NFT (metadata)**
Liste de propriétés pour la fiche NFT :
- Série : [Congo / Soleil / Géo-mathématique / ...]
- Fréquence : [code ou note]
- Élément : [feu / eau / terre / air / éther]
- Intention : [mot-clé initiatique]
- Édition : [1/1 ou édition limitée]

## Mode 2 : /contenu livre

### Étape 1 — Cadrage du passage

```
Pour quel livre ou projet d'écriture ?
(Ex : "livre sur les énergies élémentaires", "cheminement initiatique Congo", "géo-mathématique sacrée")

Quel est le sujet de ce passage ?
Donne-moi l'idée centrale en une phrase.

Longueur souhaitée :
a) Court (300-500 mots) — une entrée, un flash
b) Moyen (800-1200 mots) — un chapitre court
c) Long (2000+ mots) — chapitre complet
```

### Étape 2 — Rédaction

Rédiger dans le style initiatique littéral de Njaho :
- Commencer par une affirmation ou une image concrète, jamais une question rhétorique
- Développer par strates : le visible, le caché, le transmis
- Conclure par une ouverture, pas une conclusion fermée

Proposer d'enregistrer dans `context/import/livre-passage-[date].md`.

## Mode 3 : /contenu post

### Étape 1 — Contexte

```
Pour quelle plateforme ?
a) Instagram (visuel + légende — max 2200 caractères)
b) X / Twitter (max 280 caractères ou thread)
c) LinkedIn (professionnel — max 3000 caractères)

Quel est le sujet ?
(Ex : "lancement d'une nouvelle oeuvre", "réflexion sur la géo-mathématique", "avancement du projet NFT")
```

### Étape 2 — Générer le post

**Instagram :**
- Première ligne accrocheuse (visible avant "voir plus")
- Corps dans le style Njaho
- 5 à 10 hashtags ciblés (art africain, NFT, spiritualité, géo-mathématique)
- Call to action sobre

**X / Thread :**
- Tweet d'accroche (1er tweet)
- 3 à 5 tweets de développement
- Tweet de conclusion avec lien ou appel

**LinkedIn :**
- Angle professionnel ou entrepreneurial
- Histoire courte + insight
- Invitation à la discussion

## Mode 4 : /contenu site

### Étape 1 — Section à rédiger

```
Quelle section du site artistique ?
a) Page d'accueil (présentation de l'univers)
b) Page "À propos" (qui est Njaho)
c) Description d'une série (Congo / Soleil / Géo-mathématique)
d) Page de vente / collection NFT
e) Manifeste artistique
```

### Étape 2 — Rédaction

Texte optimisé pour le web : phrases courtes, paragraphes aérés, style initiatique préservé.
Format HTML-ready si demandé (pour intégration WordPress / Hostinger).

## Mode 5 : /contenu pitch

Générer un pitch de présentation de l'univers artistique de Njaho.

Versions disponibles :
- **30 secondes** : pour une rencontre rapide ou une bio de marketplace
- **3 minutes** : pour une présentation complète
- **Écrit** : pour un dossier ou une application NFT

Structure du pitch :
1. Qui je suis (en une phrase, pas un CV)
2. Ce que je crée (la série, la démarche)
3. Pourquoi ça compte (la tradition, la transmission)
4. Ce que je propose (l'oeuvre, la collection, le projet)

---

## Règles générales

- Ne jamais produire un texte générique sur "l'art africain" ou "la spiritualité" sans ancrage dans le vécu et la démarche spécifique de Njaho
- Toujours demander des détails concrets avant de rédiger : une formule, une couleur, une énergie, une intention
- Les textes NFT doivent donner envie de posséder l'oeuvre, pas seulement de la regarder
- Pour les livres : respecter la longueur minimale de 2000 mots par chapitre si demandé
- Proposer systématiquement de sauvegarder dans `context/import/`
- Ajouter dans HISTORY.md si un contenu majeur a été produit
