## Mode 1 : /crypto dca

### Contexte de Njaho

Stratégie DCA (Dollar Cost Averaging) : investissement régulier à intervalle fixe,
indépendamment du prix, pour lisser le coût d'entrée sur le long terme.

### Ce que tu dois faire

Demander :

```
Pour ton DCA du moment :

1. Sur quels actifs investis-tu ? (BTC, ETH, autres ?)
2. Quelle est ta fréquence habituelle ? (hebdo, mensuel)
3. Tu es sur quelle plateforme ? (Binance, Coinbase, autre)
4. Tu veux investir maintenant ou évaluer le timing ?
```

Puis analyser et recommander :

**Signal DCA du moment**
- Tendance court terme (haussière / baissière / neutre)
- Niveau de prix vs moyennes historiques (survendu / surévalué / neutre)
- Recommandation : investir maintenant / attendre / renforcer

**Raisonnement**
Explication en 3-4 lignes sans jargon inutile.

**Rappel stratégique**
Le DCA n'est pas du timing de marché. Si l'horizon est long terme (3-5 ans+),
la régularité prime sur le "bon moment". Rappeler ce principe si Njaho hésite.

## Mode 3 : /crypto portfolio

Demander :

```
Pour faire le bilan de ton portfolio :

Liste tes positions actuelles :
- Actif, quantité, prix d'achat moyen
Exemple : "BTC 0.05 à 45 000$, ETH 0.8 à 2 800$"

Tu veux aussi inclure tes NFT dans le bilan ?
```

Calculer et présenter :

**Bilan portfolio**
- Valeur actuelle vs valeur d'achat
- Performance globale en % et en valeur
- Actif le plus performant / moins performant
- Recommandation DCA : renforcer quoi en priorité

Proposer d'enregistrer dans `context/import/crypto-portfolio-[date].md`.

## Mode 5 : /crypto veille

Recherche web ciblée sur :
- Réglementation crypto (France / Europe / Martinique)
- Nouveautés NFT et art numérique
- Évolutions BTC / ETH
- Outils DCA et automatisation d'investissement

Format : 3 à 5 news maximum, une ligne chacune, source entre parenthèses.

---

## Règles générales

- Ne jamais donner de conseil financier ferme : toujours formuler en "signal" ou "observation"
- Rappeler systématiquement que le DCA long terme réduit le risque sans éliminer la volatilité
- Relier les opportunités NFT aux séries artistiques concrètes de Njaho (Congo, Soleil, Géo-math)
- Si Njaho demande un avis tranché ("j'achète ou pas ?"), donner une analyse honnête avec les risques
- Proposer de sauvegarder les bilans portfolio dans `context/import/`
- Ne pas inventer des prix : toujours chercher les données en temps réel via WebSearch
