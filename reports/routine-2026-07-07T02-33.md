## What Remains / Next Ideas

### Missing API key

`ANTHROPIC_API_KEY` is absent from `.env` — the `aigen` command (Gemini/Nano Banana integration) is dormant. Adding the key would unlock prompt-to-stencil generation.

### Potential next motifs / diversification

All 8 sacred generators and 8 plot scribes are functional. Currently the routine always picks the same 4 scribes (metatron, mandala, harmonograph, spirograph). Rotating through the unused ones would add variety:
- `sacred enneagram`, `sacred wheel --cs 3`, `sacred seal`, `sacred spiral`
- `plot rose`, `plot lissajous`, `plot phyllotaxis`, `plot attractor`

### Stencil output quality

The trace pipeline uses `--max-side 360` by default (Cricut-friendly resolution). For a hi-def gallery view, adding `--max-side 1400` to trace/stencil commands would sharpen the SVG outlines — currently the stencil is vectorised at 360 px then scaled up.

### Series features (unused)

The `trace` command supports `--stylize`, `--kaleidoscope`, `--modulate`, and `--morph-with` — none have been exercised in a routine run. A kaleidoscope variant of the forest trace would make a compelling daily artefact.

### `turbines` and `turbines-exact` render works

These exist in the registry but have never appeared in a routine run. Worth adding to the rotation alongside `forest` and `horse`.
