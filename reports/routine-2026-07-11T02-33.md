## What Remains / Next Ideas

The routine has been running cleanly for weeks. The test suite is stable. The main growth opportunities:

1. **Implement `sacred mandala`** — a configurable mandala generator with `--petals` and `--rings` parameters. Natural extension of `generate_flower`, using concentric petal rings with angular symmetry.

2. **Implement `sacred metatron`** — Metatron's Cube (Flower of Life + connecting lines to form the 13-circle pattern). Shares the hexagonal lattice logic already in `generate_flower`.

3. **Implement `plot harmonograph`** — parametric Lissajous-variant using damped pendulum equations. Pure numpy/SVG, no matplotlib dependency.

4. **Implement `plot spirograph`** — hypotrochoid/epitrochoid family curves. Two parameters (R, r, d) produce the classic spirograph shapes.

5. **Implement `render forest`** — a procedurally generated forest scene (recursive tree branching with L-system or random walk). This would give the trace step a more organic input than the horse silhouette.

6. **Add `--mode silhouette` to `sacred flower`** — currently only `sri-yantra` supports silhouette mode. A dark-background Flower of Life with white strokes would be gallery-ready.

7. **Gallery index page** — generate an HTML index of all SVGs so the gallery can be browsed visually.
