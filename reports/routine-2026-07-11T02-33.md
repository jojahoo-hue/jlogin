## What Remains / Next Ideas

The library has matured significantly. All originally-planned generators are now implemented.

1. **`render turbines` and `render turbines-exact`** -- two render works exist beyond horse and forest; generate stencils for them in a future run.

2. **Negative-mode stencils** -- all sacred/plot commands support `--mode negative` (adds a frame + auto-bridges for isolated islands). A run using negative mode would produce fully Cricut-ready cut files.

3. **`aigen` integration** -- `aigen.py` is dormant pending a Gemini API key. Once wired up, prompt-to-SVG (Nano Banana pipeline) would be the project's most distinctive feature.

4. **`sacred enneagram`, `sacred seal`, `sacred spiral`** -- not yet exercised by the routine; add to future runs.

5. **Trace variations** -- `trace` now supports `--stylize`, `--kaleidoscope`, `--modulate`, `--morph-with`. A 6-fold kaleidoscope trace of the forest would make a striking stencil.

6. **Gallery index** -- with 80+ SVGs accumulated, an auto-generated HTML index would make the gallery browsable.
