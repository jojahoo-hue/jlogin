## What Remains / Next Ideas

1. **Expand the design library** — the spec references `metatron`, `mandala`, `harmonograph`, `spirograph`, and `forest` as first-class generators. Implementing them is the clearest growth path for the project.

2. **Mandala generator with petals/rings params** — `sacred mandala --petals N --rings R` would be a natural complement to `flower` and `sri-yantra`, enabling parameterised symmetry exploration.

3. **Harmonograph** — damped Lissajous figure with four sine waves; produces organic pendulum-style curves ideal for Cricut stencils.

4. **Spirograph** — hypotrochoid/epitrochoid curve generator; another classic parametric motif absent from the current plot library.

5. **Metatron's Cube** — extends `flower` by overlaying the 13 circles with the 78 connecting line segments; high value for sacred geometry stencil work.

6. **Outline mode for `flower` and `sri-yantra`** — today both were generated in `silhouette` mode. Producing an `outline` variant alongside gives two distinct Cricut cut layers per design.

7. **SVG optimisation** — large 1400px SVGs could be run through `scour` or `svgo` as a post-processing step to reduce file size before Cricut import.

8. **CI/CD** — no GitHub Actions workflow exists yet; adding a simple test-and-lint job on push would guard the codebase.
